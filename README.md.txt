# Library Management System

## Project Overview
This project is a SQL-based Library Management System designed to manage books, authors, members, and borrow/return transactions.

## Database Schema
The database schema is defined in the `library_management.sql` file. This file contains the SQL commands to create the necessary tables and insert sample data.

## Tables
- `Books`: Stores book information.
- `Authors`: Stores author information.
- `Members`: Stores member information.
- `Loans`: Stores borrow/return transactions.
